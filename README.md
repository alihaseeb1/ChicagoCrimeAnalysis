To run locally:
1. go here and download jdk 11 : https://jdk.java.net/archive/
2. Go to C:\Program Files and create a new folder Java
3. unzip downloaded java sdk file and copy jdk-11.0.2 to the C:\Program Files\Java
4. edit sys path and add a new variable called JAVA_HOME and put value as C:\Program Files\Java\jdk-11.0.2
5. edit Path variable add new value C:\Program Files\Java\jdk-11.0.2\bin
6. open up the jupyter notebook
7. create a new venv using : python -m venv venv
8. run pip install -r requirements.txt
9. The main.ipynb has the code, don't edit it recklessly you can make another file or if editing make sure the other codes run and are not broken
