To run locally:
0. follow this video to setup the spark : https://www.youtube.com/watch?v=OmcSTQVkrvo
1. go here and download jdk 11 : https://jdk.java.net/archive/
2. create a new venv using : python -m venv venv
3. run pip install -r requirements.txt
4. The main.ipynb has the code, don't edit it recklessly you can make another file or if editing make sure the other codes run and are not broken
